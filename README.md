# Pro Utility Suite | Multi-Purpose Dashboard

Pro Utility Suite is a centralized hub of essential calculation tools designed for speed and accuracy. This project demonstrates advanced JavaScript logic, including unit conversion algorithms, theme switching, and automated report generation.

## 🚀 Key Features

### 1. Advanced Converters
- **Temperature:** Seamlessly switch between Celsius and Fahrenheit.
- **Length & Distance:** Convert units like Meters, Kilometers, Inches, and Miles.
- **Currency (Top 10):** Real-time calculation for major global currencies.

### 2. Professional Marksheet Generator
- **Logic:** Automated calculation of total marks, percentage, and grading.
- **Official Layout:** Generates a structured table with pass/fail status and final remarks.
- **Dynamic UI:** Results are rendered instantly into the DOM using modular JS functions.

### 3. User Experience (UX) Features
- **Theme Toggle:** Fully functional Dark Mode and Light Mode for comfortable viewing.
- **Responsive Grid:** A modern dashboard layout that adapts to mobile, tablet, and desktop screens.
- **Interactive Feedback:** Visual indicators for errors (e.g., "Enter Value") and success states.

## 🛠️ Technical Stack
- **HTML5:** Semantic dashboard architecture.
- **CSS3:** - CSS Variables for dynamic theme switching.
  - Grid & Flexbox for layout management.
  - Custom animations for UI elements.
- **JavaScript (ES6+):** - Complex conditional logic for unit conversions.
  - DOM Manipulation for real-time data rendering.
  - Event Listeners for theme control.

## 📂 Project Structure
- `index.html`: Main dashboard shell and tool sections.
- `style.css`: Core styling with dark/light mode logic.
- `script.js`: Calculation algorithms and UI controller.
